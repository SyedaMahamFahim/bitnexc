# Installation

## Node Version
NodeJS - 14

## Build Setup

``` bash
# install dependencies
npm i

# serve with hot reload at localhost:8080
npm install 

# build for production with minification
npm run build

# after creating a build . place a dist folder in your server file also set backend api paths .. 

```
